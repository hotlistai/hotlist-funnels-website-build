# Hotlist Funnels Website

Forging high-velocity conversion systems. Speed over sprawl. Clarity over complexity.

## Project Details
- **GitHub Repository**: [hotlistai/hotlist-funnels-website-build](https://github.com/hotlistai/hotlist-funnels-website-build)
- **Framework**: Next.js (App Router)
- **Database**: Supabase
- **Styling**: Tailwind CSS + Framer Motion
- **Deployment**: Cloudflare Workers / Pages via Wrangler

## Getting Started

First, run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Architecture

This project is built for maximum conversion speed and lead intelligence. It features:
- Industry-specific landing page templates
- Integrated revenue recovery calculator
- Direct CRM sync via Supabase
- Embedded Calendly booking flows
- Mobile-optimized lead capture systems

## Deployment

Deploy to Cloudflare:
```bash
npm run deploy
```
